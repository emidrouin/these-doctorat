"Je termine avec cet alexandrin qui n'est pas de moi: // Où l'insecte doré sur le tronc géant rôde. // Beaucoup de gens de ce pays ont ri de cet alexandrin pondu par un garçon de ce pays. Mille Milles le trouve à son goût. Doré est le contraire de rôde, d'une certaine façon. De toute façon, les deux mots sont formés des mêmes lettres. Enfin... Ceux qui sont intelligents comprennent, et apprécieront. D'ailleurs pour une mouche, un tronc est extrêmement géant. Je n'écris pas pour l'imbécilité mais pour la postérité."
