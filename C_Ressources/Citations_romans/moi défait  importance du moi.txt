"moi défait" / importance du "moi"
"Le bas du bantam c'est moi défait, c'est moi descendu en image de chose vécue pour vrai. // Le moi du bas où je ne veux pas aller, le moi bas que je ne veux pas voir, le moi bas que je ne veux pas montrer. // Le moi-là dont je dis tout le temps, c'est pas moi, c'est bien moi. // Le nommer c'est le descendre, le décrire c'est le descendre. // En parler, c'est le dénoncer. // C'est le moi du bantam dévoilé, c'est le moi du bantam révélé, c'est le moi dans le six énoncé dans le bantam tel qu'il est." (NS, 87-88) #métaphore, aussi #mise_en_scène_écriture
